# 잔향

감정으로 남기는 음악 기록장.

잔향은 음악을 듣고 남은 감정을 조용히 기록하는 작은 웹 앱입니다. 곡을 로컬 DB에서 검색하거나 직접 입력하고, 감정 태그와 짧은 메모를 남길 수 있습니다.

## 첫 버전 범위

- 홈
- 음악 로그 작성
- 로컬 DB 곡 검색
- 직접 곡 입력
- 감정 태그
- 내 로그 목록
- 로그 상세

## 제외한 것

- 음악 스트리밍
- 추천
- 플레이리스트 생성
- 좋아요, 댓글, 팔로우, 랭킹

## 실행

```bash
npm run dev
```

기본 주소는 `http://localhost:3000`입니다. 포트를 바꾸려면 `PORT=4000 npm run dev`처럼 실행합니다.

## 확인

```bash
npm run build
npm run typecheck
npm run lint
npm test
```

## 데이터

초기 곡 데이터는 `data/db.seed.json`에 있습니다. 앱을 처음 실행하면 로컬 작업용 DB인 `data/db.json`이 생성되며, 이 파일은 git에 포함하지 않습니다.
